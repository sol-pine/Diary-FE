# 매일의 기분을 기록하는 mood diary
TDD 방법론을 공부하고 적용해보기 위해 개발한 토이 프로젝트입니다.

### Installation
```javascript
$ npm install
$ npm start
```
### 구현 기능
* 회원가입 및 로그인
* 오늘 기분 기록
* 기분 수정
* 기분 삭제
